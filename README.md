# Perch Gallery

Community catalog for [Perch](https://github.com/wouterdp/perch-2) â€” a cross-platform dotfiles and application settings manager.

Served via GitHub Pages at `https://wouterdp.github.io/perch-gallery/`.

## Structure

```
catalog/
  index.yaml              # Master index of all entries
  apps/<id>.yaml           # Application definitions
  fonts/<id>.yaml          # Font definitions
  tweaks/<id>.yaml         # Windows registry tweak definitions
```

**Fonts (5):** Cascadia Code NF, JetBrains Mono NF, Fira Code, Hack NF, Meslo NF

**Tweaks (15):** File extensions, hidden files, dark mode, taskbar alignment, web search, context menu, sticky keys, compact mode, Explorer default, Copilot button, widgets, chat icon, advertising ID, mouse acceleration, long paths
